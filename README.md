# LinkTalkProject

## 프로젝트 개요
LinkTalkProject는 **YouTube API를 활용해 자동으로 영상을 업로드하고, 제휴 링크를 설명에 삽입하는 기능**을 구현하는 오픈소스 프로젝트입니다.  
학습과 협업을 통해 기능을 완성하고, 코드 품질을 개선하는 것을 목표로 합니다.

## 현재 문제 상황
- main.py 실행 시 문자열 처리 오류 발생 (예: "unterminated string literal")
- GitHub Actions 실행 시 main.py와 Update main.py 파일 충돌 문제
- 서비스 계정 인증 및 GitHub Secrets 관리 최적화 필요

## 제가 원하는 방향
- 안정적으로 YouTube API를 통해 영상을 자동 업로드할 수 있도록 코드 구조 개선
- GitHub Actions 워크플로우에서 main.py만 정상적으로 실행되도록 환경 정리
- 서비스 계정 인증 및 Secrets 관리 방식을 개선해 보안성과 자동화를 동시에 확보
